# 📊 Customer Segmentation & Satisfaction Analytics Dashboard (Power BI)

## 📌 Project Overview

This project analyzes customer demographic, financial, and satisfaction data to uncover business insights that can help improve customer experience, targeting strategies, and financial decision-making.

The dashboard provides a structured business view across:

- Executive Summary
- Customer Demographics
- Satisfaction Analysis
- Financial & Asset Insights

---

## 🎯 Business Objectives

The dashboard answers key business questions such as:

- Who are the most valuable customer segments?
- How does income influence satisfaction?
- Which job categories show higher customer satisfaction?
- Do loan customers have lower satisfaction?
- Which states have the highest concentration of customers?

---

## 🗂 Dataset Description

The dataset includes the following features:

- Customer_Age  
- Gender  
- Education_Level  
- Marital_Status  
- Customer_Job  
- Income  
- Dependent_Count  
- House_Owner  
- Car_Owner  
- Personal_loan  
- Cust_Satisfaction_Score  
- State_cd  
- Client_Num

## 🛠 Tools & Technologies Used

- Power BI
- Power Query (Data Cleaning)
- DAX (Calculated Columns & Measures)
- Data Modeling
- Interactive Slicers & Drilldowns

---

## 📐 Data Preparation Steps

1. Removed duplicate records using Client_Num.
2. Cleaned null and "Unknown" values.
3. Corrected data types (Income → Decimal, Satisfaction → Whole Number).
4. Created calculated columns:
   - Age Group
   - Income Group
   - Satisfaction Level
5. Created KPI measures:
   - Total Customers
   - Average Income
   - Average Satisfaction

---

## 📊 Dashboard Structure

### 1️⃣ Executive Overview
- Total Customers (KPI)
- Average Income (KPI)
- Average Satisfaction Score (KPI)
- Customers by Gender
- Customers by Age Group
- State-wise Customer Distribution

### 2️⃣ Customer Demographics
- Education Level Distribution
- Job Category Analysis
- Dependents Distribution
- House Ownership Analysis

### 3️⃣ Satisfaction Analysis
- Satisfaction by Age Group
- Satisfaction by Income Group
- Satisfaction by Job Category
- Loan vs Satisfaction Comparison

### 4️⃣ Financial & Asset Insights
- Income vs House Ownership
- Income vs Car Ownership
- Income by Education Level
- Loan Distribution by State

---

## 📈 Key Insights

- Middle-aged customers show relatively higher satisfaction.
- High-income customers are more likely to own houses and cars.
- Customers with personal loans show slightly lower satisfaction scores.
- Certain job categories demonstrate consistently higher satisfaction.
